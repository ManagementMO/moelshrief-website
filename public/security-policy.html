Security Policy

If you discover a security vulnerability within this website, please report it to me through GitHub.

I take security seriously and will respond to all legitimate security reports promptly.

Please include the following information in your report:

  • Description of the vulnerability
  • Steps to reproduce the issue
  • Potential impact of the vulnerability
  • Any suggested fixes (if available)

I will acknowledge receipt of your report within 48 hours and provide a more detailed response within 7 days.
